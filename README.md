# Nuxtedo-Moon

**Nuxtedo-Moon** is a *portfolio/blog Nuxt template* designed using Nuxt-Content, Tailwind CSS and daisyUI.

## Nuxt Standard Information

### Installation

Make sure to install dependencies after cloning the original repository locally:

```bash
pnpm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

### Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
